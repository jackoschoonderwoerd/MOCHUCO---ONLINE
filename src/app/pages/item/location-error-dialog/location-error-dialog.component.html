In order for The Mochuco app to work properly, both your {{device}} and {{browserName|titlecase}} need access to your location.

intructions for {{device}}

  1. On your device: Go to settings
  2. Select "Privacy"
  3. "Location Services" should be checked.
  4. In the same window, make sure that {{browserName|titlecase}} is set to "While Using the App"
intructions for {{device}}
  1. On your {{device}}: Go to settings
  2. Select "Security and pivacy"
  3. Select Location services
  4. "Access to my location" should be checked
  5. After this your {{browserName}} may ask for permission
  1. On your {{device}}: Go to settings
  2. Select "Apps & notifications"
  3. Select "Apps"
  4. Select {{browserName}}
  5. Select "Permissions"
  6. Make sure that "Your location" is checked
